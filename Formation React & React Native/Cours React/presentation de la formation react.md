# 📘 Formation React avec Vite

## 🎯 Objectif
Ce module de formation vous apprendra à développer une application web avec **React** et **Vite** en prenant comme projet fil rouge : **The Todo List App**.
---

## 📅 Programme détaillé

### 🏁 **Leçon 1 : Introduction à React et mise en place du projet**
- Présentation de React : avantages, différences avec d'autres frameworks.
- Introduction à Vite : pourquoi l'utiliser ?
- Présentation de ES6 et son intérêt.
- Installation de l'environnement de développement.
- Création du projet **The Todo List App** avec Vite.
- Structure d'un projet React.
- Premier composant : affichage d'un simple texte.

**📝 Exercice :** Afficher un message de bienvenue dans l'application.

---

### 🎨 **Leçon 2 : Composants, State et Props**
- Présentation des composants de base et du JSX.
- Différence entre composants fonctionnels et class-based.
- Utilisation du `useState` pour gérer l'état local.
- Passage des données entre composants avec les props.
- Création du premier composant réutilisable : **TodoItem**.

**📝 Exercice :** Créer un composant **TodoItem** qui affiche une tâche avec un bouton pour la supprimer.

---

### 🎨 **Leçon 3 : Styles et mise en forme des composants**
- Différentes façons de styler les composants :
  - CSS modules
  - CSS-in-JS (styled-components)
  - Tailwind CSS
- Gestion des couleurs et thèmes.
- Flexbox et Grid en CSS.
- Ajout d'icônes avec `react-icons`.

**📝 Exercice :** Styliser l'application avec des couleurs et des styles bien organisés.

---

### 📜 **Leçon 4 : Listes et gestion des événements**
- Rendu de listes avec la méthode `.map()`.
- Gestion des événements avec `onClick`.
- Ajout de nouvelles tâches avec des formulaires contrôlés.
- Mise à jour de l'état global.

**📝 Exercice :** Ajouter une tâche à la liste avec un champ de saisie.

---

### 🚀 **Leçon 5 : Navigation avec React Router**
- Installation et configuration de `react-router-dom`.
- Création d'une navigation entre différentes pages.
- Passage de paramètres entre routes.
- Utilisation des hooks de navigation.

**📝 Exercice :** Créer une page de détails pour afficher une tâche spécifique.

---

### 🗄️ **Leçon 6 : Gestion du contexte et stockage des données**
- Introduction à `Context API` pour la gestion globale de l'état.
- Mise en place d'un `TodoContext`.
- Introduction à LocalStorage.
- Sauvegarde et récupération des tâches stockées localement.

**📝 Exercice :** Sauvegarder et restaurer les tâches avec LocalStorage.

---

### 🌐 **Leçon 7-8 : Hooks avancés et gestion des effets de bord**
- Utilisation de `useEffect` pour gérer les effets de bord.
- Gestion des dépendances dans `useEffect`.
- Utilisation de `useReducer` comme alternative à `useState`.
- Optimisation des performances avec `useMemo` et `useCallback`.
- Requêtes API avec `fetch` et gestion des états de chargement.

**📝 Exercice :** Récupérer des tâches depuis une API et les afficher.

---

### 🎬 **Leçon 9 : Animation et interactions avancées (Optionnel)**
- Introduction aux animations CSS.
- Utilisation de bibliothèques comme `framer-motion`.
- Ajout d'animations sur l'ajout/suppression de tâches.
- Transitions de page fluides.

**📝 Exercice :** Animer la suppression d'une tâche.

---

### 🚢 **Leçon 10 : Déploiement de l'application (Optionnel)**
- Build de production avec Vite.
- Déploiement sur Vercel ou Netlify.
- Optimisation des performances.

**📝 Exercice :** Déployer votre application Todo List.

---

## 🎓 Prérequis
- Connaissance de base en HTML, CSS et JavaScript.
- Node.js installé (version 16 ou supérieure recommandée).
- Un éditeur de code (VS Code recommandé).

---

## 🛠️ Outils et technologies utilisés
- **React** : Bibliothèque JavaScript pour construire des interfaces utilisateur.
- **Vite** : Outil de build ultra-rapide pour les applications web modernes.
- **React Router** : Pour la navigation entre les pages.
- **LocalStorage** : Pour persister les données côté client.
- **CSS Modules / Styled Components** : Pour le styling.
