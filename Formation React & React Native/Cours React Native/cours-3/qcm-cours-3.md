# QCM - Cours 3 : Styles et mise en forme avec Flexbox

## Questions

**Question 1:** Quel est le système de mise en page principal en React Native ?
- A) Grid
- B) Float
- C) Flexbox
- D) Table

**Question 2:** Quelle est la valeur par défaut de `flexDirection` en React Native ?
- A) row
- B) column
- C) row-reverse
- D) column-reverse

**Question 3:** Quelle propriété contrôle l'alignement des éléments sur l'axe principal ?
- A) alignItems
- B) justifyContent
- C) flexAlign
- D) mainAlign

**Question 4:** Quelle propriété contrôle l'alignement des éléments sur l'axe secondaire ?
- A) justifyContent
- B) alignItems
- C) secondaryAlign
- D) crossAlign

**Question 5:** Si `flexDirection` est `row`, quel est l'axe principal ?
- A) Vertical
- B) Horizontal
- C) Diagonal
- D) Circulaire

**Question 6:** Quelle valeur de `justifyContent` centre les éléments ?
- A) flex-start
- B) flex-end
- C) center
- D) middle

**Question 7:** Quelle valeur de `justifyContent` répartit l'espace sans espace aux extrémités ?
- A) space-around
- B) space-evenly
- C) space-between
- D) space-equal

**Question 8:** À quoi sert la propriété `flex` ?
- A) À créer des animations
- B) À définir comment un élément occupe l'espace disponible
- C) À changer la couleur
- D) À ajouter des bordures

**Question 9:** Quelle valeur d'`alignItems` étire les éléments pour remplir l'espace ?
- A) fill
- B) stretch
- C) expand
- D) full

**Question 10:** En React Native sur le web, quelle est la différence majeure concernant Flexbox ?
- A) Il n'y a aucune différence
- B) L'axe principal est horizontal (row) par défaut sur le web, vertical (column) sur React Native
- C) Flexbox ne fonctionne pas sur React Native
- D) On ne peut pas utiliser justifyContent sur le web

---

## Réponses

1. **C** - Flexbox
2. **B** - column
3. **B** - justifyContent
4. **B** - alignItems
5. **B** - Horizontal
6. **C** - center
7. **C** - space-between
8. **B** - À définir comment un élément occupe l'espace disponible
9. **B** - stretch
10. **B** - L'axe principal est horizontal (row) par défaut sur le web, vertical (column) sur React Native
