# QCM - Cours 1 : Introduction à React Native et mise en place du projet

## Questions

**Question 1:** Qu'est-ce que React Native ?
- A) Un framework pour créer des sites web
- B) Un framework pour créer des applications mobiles natives en utilisant JavaScript
- C) Un langage de programmation mobile
- D) Un système d'exploitation mobile

**Question 2:** Quel est le principal avantage du développement cross-platform avec React Native ?
- A) Il ne fonctionne que sur iOS
- B) Un même code peut fonctionner sur iOS et Android
- C) Il est plus lent que le développement natif
- D) Il nécessite deux équipes différentes

**Question 3:** Quelle est la différence principale entre React Web et React Native concernant les éléments d'interface ?
- A) React Native utilise des balises HTML comme `<div>` et `<p>`
- B) React Native utilise des composants comme `View` et `Text`
- C) Il n'y a aucune différence
- D) React Native utilise uniquement du CSS

**Question 4:** Qu'est-ce qu'Expo ?
- A) Un système d'exploitation
- B) Un ensemble d'outils qui facilite le développement avec React Native
- C) Un langage de programmation
- D) Un navigateur web

**Question 5:** Quel est l'avantage principal d'utiliser Expo ?
- A) Il est payant
- B) Configuration simplifiée, pas besoin de Xcode ou Android Studio au démarrage
- C) Il fonctionne uniquement sur Android
- D) Il ne supporte pas les API natives

**Question 6:** En ES6, quelle est la différence entre `let` et `const` ?
- A) Aucune différence
- B) `let` crée une variable modifiable, `const` une constante non modifiable
- C) `const` est plus rapide que `let`
- D) `let` ne peut être utilisé que dans les fonctions

**Question 7:** Quel composant React Native affiche du texte ?
- A) `<div>`
- B) `<p>`
- C) `<Text>`
- D) `<Paragraph>`

**Question 8:** Comment définit-on les styles en React Native ?
- A) Avec des fichiers CSS externes
- B) Avec des objets JavaScript via `StyleSheet`
- C) Avec des classes CSS
- D) Avec du HTML inline

**Question 9:** Qu'est-ce que le Hot Reloading en React Native ?
- A) Une technique pour chauffer le téléphone
- B) La possibilité de visualiser instantanément les modifications sans redémarrage complet
- C) Un système de refroidissement
- D) Un outil de débogage

**Question 10:** Quelle bibliothèque est couramment utilisée pour la navigation en React Native ?
- A) React Router
- B) React Navigation
- C) Express Router
- D) Angular Router

---

## Réponses

1. **B** - Un framework pour créer des applications mobiles natives en utilisant JavaScript
2. **B** - Un même code peut fonctionner sur iOS et Android
3. **B** - React Native utilise des composants comme `View` et `Text`
4. **B** - Un ensemble d'outils qui facilite le développement avec React Native
5. **B** - Configuration simplifiée, pas besoin de Xcode ou Android Studio au démarrage
6. **B** - `let` crée une variable modifiable, `const` une constante non modifiable
7. **C** - `<Text>`
8. **B** - Avec des objets JavaScript via `StyleSheet`
9. **B** - La possibilité de visualiser instantanément les modifications sans redémarrage complet
10. **B** - React Navigation
